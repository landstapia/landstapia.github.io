# Landstapia GitHub Pages

A collection of web-based utilities and personal projects hosted via GitHub Pages.

## 🚀 Live Demo
You can view the live site here: [landstapia.github.io](https://landstapia.github.io/)

## 🛠️ Included Tools
This repository contains several standalone web applications:

* **Prompter / PrompterV2**: A web-based teleprompter for lyrics.
* **Clock**: A simple, clean digital clock interface which can also be used for watermarks.
* **Watermark**: A utility for adding watermarks to church services automatically with formatted information.

## 💻 Tech Stack
The projects in this repository are built using standard web technologies:
* **HTML5**
* **CSS3** (with custom stylesheets)
* **JavaScript** (utilizing jQuery for DOM manipulation)
